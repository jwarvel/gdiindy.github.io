Computer Programs

  • Computer programs are everywhere.
    • Run parts of your car.
    • Let you check your email.
    • Manage elevators.
    • Turn street lights on or off.
    • Tell a robot on Mars what to do.
    • Underlie that favorite app on your smartphone.
  • But what are they, really?

What is a Computer Program?

At its core, a computer program is
a way of getting something done.

  • One of the ways to accomplish "getting something done" is a list of instructions that tells the computer exactly what to do and allows interaction with human beings.
  • Those instructions have to be written in a "language" that the computer can understand.

Programming Languages

A programming language is, basically, a set of strings (letters, numbers, commands) that are turned into machine code that the computer can understand.

The language you use for a program depends on a number of things:

  • How the program will be used.
  • What you want the program to do.
  • What languages you know.

Programming Languages

  • Computers don’t exactly understand programming languages (such as Java or Python). They need to be transformed into the computer language that they can understand.
  • That computer language is strings of 1’s and 0’s-- binary numbers that most people can’t read easily-- to be able to do anything with them.
    • For example, the letter 'A' in a computer has the binary representation: 01000001.
  • Despite being able to decipher those unintelligible strings of 1’s and 0’s, computers are actually pretty dumb.

Programming, Modeling, and Querying Languages

For example:

  • To develop websites, you might use HTML, JavaScript, Python, Java, PHP.
  • To create databases and move information in and out of them, you might use SQL with MySQL, SQL Server, or Dbase.
  • To write applications like Microsoft Word or Adobe Acrobat, you might use C++, Java, Visual Basic

Computers are dumb

  • Computers can only do what they have instructions for-- and they take those instructions VERY literally.
  • As a result, you have to tell them every single thing you want them to do and every single thing to take into consideration.
  • If you leave any instructions out (or put them in the “wrong” order), the results might not be what you expect.

Memory and Processor

When running a program, a computer uses memory and processing power. Those two are currencies for programmers.

A computer has two main types of memory. The disk storage, and its volatile operating memory: the Random Access Memory (RAM). The RAM is the memory that counts when writing a program. Variables, lists, and all the data the program needs to use are stored in the RAM.

The processor is the part of the computer that actually runs the program and executes its statements.

Why is Memory Important?

  • As opposed to disk space, RAM is quite expensive and RAM considerate programs save money on server costs.
  • Programs manipulating a lot of data can easily duplicate them and use a lot of RAM.
  • A program can leak memory, meaning that because of bad code, it holds on to data that is not needed anymore, taking up more and ore RAM until there is no more and the computer crashes.
  • Sometimes optimizing memory means adding more load to the processor.

Why is Processing Power Important?

  • It's often the single most expensive resource on a server, and the limitting factor of a program's speed.
  • A program with a bad complexity can look fine with small sets of data and tests, and then become unusable when used with larger sets.
  • A powerful state of the art server might be 5 or 10 times faster than an older model, it won't make up for programms that are billions of times slower because of larger data sets.
  • Sometimes optimizing the complexity means storing more data in memory.

I have a bunch of guinea pigs with a mix of three colors: black, brown, and white. They can have one, a mix of two, or even all three colors. I'm curious to know how many guinea pigs show each color so I'm going to come up with the three numbers:

  • Number of guinea pigs with some black
  • Number of guinea pigs with some brown
  • Number of guinea pigs with some white

  • for each guinea pig
    • if the guinea pig has some black
      • add one mark to the black post it
    • if the guinea pig has some brown
      • add one mark to the brown post it
    • if the guinea pig has some white
      • add one mark to the white post it
  • look at all three post-its when it's over for the results

Data Types

Data types are quite literally types of data that a programming language understands and is able to manipulate. They vary from a language to another, but the following ones are vastly shared accross modern languages:

  • Integer (Natural number, like 1 or 2)
  • Float (Number, like 3.456)
  • String (Text, like "I am a string")
  • Boolean (true or false)

Integers

Integer are positive or negative whole numbers.

Examples: 1, -65, 8746278467823

    Real life examples:
  • On Youtube, the number of views for a video.
  • On Amazon, the number of items of a given type in your cart, or how many copies of a book you want to order.
  • Your computer's printing utility asking you how many copies you want to print, or what page number you want to print.
  • The temperature on your house's air conditioning interface.
  • The number of likes of a post on Facebook.

Floats

Floats are positive or negative numbers.

Examples: 3.14159, 10/3, -0.2

    Real life examples:
  • In Word, the page width in inches.
  • On Amazon, an item's price or your cart's total value in dollars.
  • Your body temperature on a digital thermometer

Strings

Strings are text samples

usually represented inside quotation marks.

Examples: "Hello World!", "Jane Doe", "123"

    Real life examples:
  • On any site where you have an account, your username, email address, and password are strings.
  • The content of a tweet is a string.
  • Your computer name or WiFi network name.

Booleans

Booleans can take one of the following two values: true or false.

Examples: well...true, false ;-)

    Real life examples:
  • On most sites where you have an account, your choice of receiving the promotions by email.
  • Your computer's mute state.

Lists

A list is an advanced type that can store multiple pieces of data, often from any type, in an ordered fashion. Lists are usually represented with square braquets, holding multiple values separated by commas.

Examples: [1, 2, 3, 4, 5], ['one', 'two']

    Real life examples:
  • On your phone, the list of the permissions granted to an app.
  • On Netflix the list of movies or shows in a given category, and the list of episodes in a show.

Dictionary

A dictionary is another advanced type that can store multiple pieces of data, often from any type, as key: value pairs. Dictionaries are usually represented with curly braquets, holding multiple key: value pairs separated by commas.

Examples: {firstName: "Jane", lastName: "Doe", age: 37}, {january: 31, february: 28}

    Real life examples:
  • Your user profile on most sites will be used by the code as a dictionary.
  • On Netflix the categories of shows are keys of a dictionaries, the list of shows in a category are the value.
  • Yes, that means there can be dictionaries of lists, and also lists of dictionaries, and it can go on and on

Variables (1/3)

A variable is a storage container. It has a name, and holds a value. The value can be any type of data.

A variable needs to be declared. That's when it gets assigned a name. After being declared, it can be assigned a value. From there on, the program can use the variable's name to refer to the value that's in it.


            var username;
            username = "jane123";
            console.log(username);

Variables (2/3)

In most languages, variables can be declared and have a value assigned to at the same time


            var username = "jane123";
            console.log(username);

Some languages won't let you store values from different data types under the same variable name. This means the type a variable can store is defined during declaration.


            String username = "jane123";
            int age = 37;

Variables (3/3)

A variable can have its value reassinged as many times as necessary. When that happens, the previous value in the variable is lost and the variable name now refers to the new value.


            var username = "jane123";
            console.log(username); // will log "jane123"
            username = "a";
            console.log(username); // will log "a"
            username = 4 // this would break in a language like Java
            console.log(username + 2) // will log 6

You can also use a variable in its own assignment instruction:


            var age = 37;
            age = age + 1; // it's someone's birthday!!!
            console.log(age) // will log 38

If / Then / Else

If / Then / Else statements evaluate a condition
and take actions based on the result.

  • If the condition is true, the computer does the action or actions that are listed after the IF statement (in some languages, the "Then" is implied).
  • If the condition is false, the computer does the action or actions that are listed in the Else statement.

If/Then/Else Examples

  • Another Amazon example...
    • "IF" you're logged in, show your name and cart numbers. "ELSE", show a link to the login page.
    • "IF" you're an Amazon Prime member, you get free shipping. "ELSE", you get to pay for shipping!

Loops

A loop is a set of instructions that repeats
until a certain condition is reached.

Loops

  • A loop is a set of instructions that repeats until a certain condition is reached.
  • Two kinds of loops are For Loops and While Loops.

Why Use Loops?

  • Loops are very powerful. One primary reasons to use loops is to reduce your lines of code.
  • Depending on what you're doing, you could have hundreds of lines of code using if/then/else, or you could write a simple loop statement.

Let's Do Some Loops!

  • For this excercise, we're using Python.
  • Go to repl.it.
  • Choose 'Python'.

Let's Do Some Loops!



            papers_to_deliver = 65

            while papers_to_deliver > 0:
                print papers_to_deliver
                papers_to_deliver = papers_to_deliver - 1

            print 'Out of papers! Go home!';
  • Change the numbers to see what happens.
  • Change the operators (greater than, less than, etc.)
  • Change the string statements.

FOR Loop Examples

  • Newsletter: "FOR" each subscriber to the newsletter, send an email.
  • Amazon cart total: "FOR" each item in the cart, update the total to add the price of the item times the number of copies.

WHILE Loop Examples

  • Car alarm system when triggered: "WHILE" the engine is off, keep making noise.
  • Radio search for frequencies: "WHILE" there is no signal, increase the scanned frequency

Beware the Infinite Loop!

  • An infinite loop is a loop that will never meet the condition to stop. It will keep going until it's used up all your computer's (or server's) memory (RAM). This is bad!
  • Example for our loop:
    • If the papers_to_deliver = papers_to_deliver - 1 line was missing, it would keep looping because 65 is always greater than 0!
    • If papers_to_deliver = papers_to_deliver - 1 was changed to + 1, it would keep adding 1 to 65 forever!

From Pseudo to Real Code


          blackNumber = 0
          brownNumber = 0
          whiteNumber = 0
          for pig in guineaPigs:
              if pig.hasColor('black'):
                  blackNumber = blackNumber + 1
              if pig.hasColor('brown'):
                  brownNumber = brownNumber + 1
              if pig.hasColor('white'):
                  whiteNumber = whiteNumber + 1
          print('Number of guinea pigs with black:')
          print(blackNumber)
          print('Number of guinea pigs with brown:')
          print(brownNumber)
          print('Number of guinea pigs with white:')
          print(whiteNumber)

From Pseudo to "Real Life" Code


          colorCount = {
              "black": 0,
              "brown": 0,
              "white": 0
          }
          for pig in guineaPigs:
              for color in colorCount.keys():
                if pig.hasColor(color):
                  colorCount[color] ++
          for color in colorCount.keys():
              print('Number of guinea pigs with %s:' % color)
              print(colorCount[color])
