attr('attribute', 'value')

Sets the value of the property for all elements that matches your selector.

<a href="http://www.google.com" style="color:purple">Yahoo!</a>
$('a').attr('href', 'http://www.yahoo.com');

Changes the HTML to:

<a href="http://www.yahoo.com" style="color:purple">Yahoo!</a>
