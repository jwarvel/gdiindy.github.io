Before we use jQuery, we have to include the library just like our regular JavaScript files.

The most popular way to do this is to use a CDN.

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>

Grab the snippet:
goo.gl/iKFHs

Add the snippet to the practice file index.html. Make sure you add http: at the beginning of the src!
